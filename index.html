Bring It

• • •

The ultimate party planner! Bring It consists of Users, their Events, and the Items that each Event needs to be a party. Attendees can then add Items they plan on bringing, or claim any the host needs. All in a single page application.
The data is stored in a cloud database called Mongoose. Bring It works through a custom-built API that handles the logic and management of that data.

Tools Used


• NodeJS • Express • Webpack • MongoDb • HTML5 • CSS3 • Github • VSCode • Postman
